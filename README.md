# Infix to Prefix Converter & Evaluator

## Project Description
A robust Java Swing application designed to convert mathematical expressions from **Infix** notation to **Prefix** notation (Polish Notation) and evaluate the results. This project demonstrates the practical application of the **Stack** data structure in expression parsing and algorithmic logic.

## Key Features
- **Expression Conversion:** Converts standard Infix (e.g., `(A+B)*C`) to Prefix (e.g., `*+ABC`).
- **Mathematical Evaluation:** Provides the final numerical result of the prefix expression.
- **Support for Multi-digit Numbers:** Handles numbers larger than 9 through tokenization.
- **Interactive GUI:** User-friendly interface built with Java Swing.

## Technical Stack
- **Language:** Java
- **Library:** Java Swing (GUI), AWT
- **Core Concept:** Stack Data Structure

## How to Run
1. Ensure Java (JDK) is installed on your system.
2. Clone the repository and open it in your preferred IDE (NetBeans, IntelliJ, etc.).
3. Run the `ConverterProject.java` file to launch the application.
